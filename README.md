## ⚙️ 技术实现细节

### 动画引擎
*   **数字雨**：
    *   使用 HTML5 `<canvas>` API 绘制。
    *   逻辑：创建粒子数组，随机从字符集 `01PYTHONFLASKGITDATA` 中选取字符，通过控制 Y 轴坐标和透明度实现下落与拖尾效果。

### 响应式设计
*   **Mobile First**：
    *   宽度 < 768px 时，双栏 Grid 自动切换为单列。
    *   导航栏在移动端隐藏（保持极简），Hero 标题字号自动缩小。
    *   知识终端 Tab 保持点击交互，但适配手指触控区域。

### CSS 特性
*   **Backdrop Filter**：顶部状态栏使用了 `backdrop-filter: blur(5px)`，在滚动时产生毛玻璃效果。
*   **CSS Variables**：全站颜色与字体统一定义在 `:root` 中，方便后续主题微调。
*   **Scroll Behavior**：导航链接点击后使用 `scroll-behavior: smooth` 实现平滑滚动。
